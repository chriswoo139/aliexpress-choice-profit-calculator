# 速卖通全托管女性内衣选品与报价工具使用文档

## 1. 工具用途

本工具用于速卖通 AliExpress Choice / 全托管模式下的女性内衣类目选品、报价和上架资料整理，适合运营、采购、产品开发人员使用。

适用类目包括：

- 无痕内裤
- 运动内衣
- 文胸延长扣
- 肩带防滑扣
- 胸贴
- 塑身衣
- 其他女性内衣或配件类产品

工具可以帮助你完成：

- 判断产品是否值得打样
- 判断产品是否适合报价
- 计算全托管供货价下是否有利润
- 反推出安全报价和激进报价
- 生成英文标题、五点卖点、商品属性、尺码表和包装信息
- 检查内衣类目常见合规风险
- 导出 Excel，方便导入 ERP 或人工上架

## 2. 打开工具

如果是本地使用，在项目目录运行：

```bash
streamlit run app.py
```

如果已经部署到 Streamlit Cloud，直接打开你的公网链接即可。

进入页面后，会看到中文界面，页面主要分为：

- 数据录入区
- 产品数据表格
- 选品评分
- 利润报价
- 商品资料
- 合规检查
- 导出

## 3. 数据录入方式

工具支持两种录入方式。

### 3.1 上传 CSV / Excel

点击页面顶部的“上传 CSV / Excel”，选择你的产品表格。

建议先点击“下载空白导入模板”，按照模板字段填写后再上传。

上传后，系统会把数据导入到页面表格中，你还可以继续手动修改。

### 3.2 手动新增产品

点击“手动新增一个产品”，填写产品信息后点击“添加到表格”。

适合临时测算一个新产品，例如：

- 新看到的 1688 产品
- 供应商报价产品
- 准备打样的 SKU
- 想临时判断利润的产品

## 4. 字段填写说明

### 基础字段

| 字段 | 说明 | 示例 |
|---|---|---|
| 产品名称 | 中文产品名称，方便内部识别 | 女士无痕冰丝内裤三条装 |
| SKU | 内部 SKU 编码 | NXK-3P-001 |
| 产品类型 | 从下拉选项中选择 | 无痕内裤 |

### 成本字段

| 字段 | 说明 | 示例 |
|---|---|---|
| 采购价 RMB | 单个产品的采购价，不是整套总价 | 3.2 |
| 包装成本 RMB | 单套包装成本，例如 OPP 袋、贴标、彩卡 | 0.55 |
| 国内运费 RMB | 单套分摊的国内运费 | 0.5 |
| 质检人工 RMB | 单套分摊的质检、理货人工 | 0.2 |
| 损耗率 % | 次品、少货、返工等损耗比例 | 3 |
| 资金占用率 % | 备货、账期导致的资金成本 | 1.5 |

注意：如果是 3 条装内裤，采购价填写单条采购价，组合件数填写 3。

### 产品结构字段

| 字段 | 说明 | 示例 |
|---|---|---|
| 单品重量 g | 单个产品重量，未含外箱 | 35 |
| 组合件数 | 一套里有几件 | 3 |
| 供应商起订量 | 供应商 MOQ | 300 |
| 颜色数量 | 可选颜色数量 | 6 |
| 尺码数量 | 可选尺码数量 | 4 |
| 是否轻小件 | 是否适合轻小件物流 | 是 |

### 风险和市场字段

| 字段 | 说明 | 示例 |
|---|---|---|
| 是否容易退货 | 尺码复杂、试穿要求高、客诉多的产品选“是” | 否 |
| 是否有合规风险 | 图片或表达可能敏感时选“是” | 否 |
| 差异化卖点 | 产品与普通竞品不同的地方 | 无痕、冰丝触感、中腰、三条装 |
| 竞品最低售价 | 竞品中的低价参考 | 16.9 |
| 竞品主流售价 | 主流竞品价格，不一定是最低价 | 22.9 |
| 预估平台供货价 | 准备报给平台的供货价 | 15.9 |

竞品数据需要人工录入或通过 CSV / Excel 导入。本工具不做平台爬虫。

## 5. 选品评分怎么看

点击“选品评分”标签页。

系统会输出：

- 市场需求
- 供应价优势
- 尺码稳定性
- 差异化卖点
- 物流包装友好度
- 退货风险
- 合规风险
- 总分
- 等级
- 是否建议打样
- 是否建议报价
- 最终建议

### 等级解释

| 等级 | 分数 | 含义 |
|---|---:|---|
| A | 85 分及以上 | 优先产品，可以进入重点报价池 |
| B | 70-84.9 分 | 可以考虑，建议结合利润和供应链稳定性判断 |
| C | 55-69.9 分 | 谨慎观察，建议补充竞品、成本或卖点信息 |
| D | 55 分以下 | 不建议优先做 |

### 操作建议

| 输出 | 建议动作 |
|---|---|
| 建议打样 | 可以联系供应商拿样，重点检查面料、做工、包装 |
| 暂缓打样 | 先补充竞品价格、利润或差异化信息 |
| 建议报价 | 可以进入报价表，准备报给平台 |
| 谨慎/暂不报价 | 利润或风险不够安全，建议重新谈采购价或提高供货价 |
| 建议放弃 | 当前条件下不适合投入时间 |

## 6. 利润报价怎么看

点击“利润报价”标签页。

核心指标包括：

| 指标 | 含义 |
|---|---|
| 单件真实成本 | 做这个产品真实要付出的单套成本 |
| 平台供货价 | 当前准备报给平台的价格 |
| 单件毛利 | 平台供货价减去真实成本 |
| 毛利率 | 毛利占平台供货价的比例 |
| 最低可接受供货价 | 低于这个价格通常不建议报价 |
| 建议报价区间低 | 按目标毛利率反推的报价 |
| 建议报价区间高 | 更安全的报价 |
| 安全报价 | 利润安全垫更高的报价 |
| 激进报价 | 为了拿单可以考虑的较低报价 |
| 放弃报价 | 低于该价格建议放弃 |

### 计算公式

```text
基础成本 = 采购价 RMB × 组合件数 + 包装成本 RMB + 国内运费 RMB + 质检人工 RMB
损耗成本 = 基础成本 × 损耗率
资金占用成本 = 基础成本 × 资金占用率
单件真实成本 = 基础成本 + 损耗成本 + 资金占用成本
单件毛利 = 预估平台供货价 - 单件真实成本
毛利率 = 单件毛利 / 预估平台供货价
按毛利率反推报价 = 单件真实成本 / (1 - 目标毛利率)
```

### 报价使用建议

- 如果想稳妥：优先参考“安全报价”
- 如果想提高中标概率：可以参考“激进报价”
- 如果平台压价低于“放弃报价”：建议不要接
- 如果毛利率为负：说明供货价低于真实成本，需要重新谈采购价或放弃

## 7. 商品资料怎么用

点击“商品资料”标签页。

系统会自动生成：

- 英文标题
- 英文五点卖点
- 英文详情页描述
- 商品属性
- 材质说明
- 尺码表
- 包装信息
- 图片拍摄提示词
- 主图提示词
- 详情页图片文案

这些内容可以用于：

- ERP 商品资料初稿
- 人工上架资料
- 供应商拍图需求
- 美工详情页文案
- 平台报价资料补充

注意：自动生成内容是基础稿，上架前建议人工检查标题、材质、尺码和敏感词。

## 8. 合规检查怎么看

点击“合规检查”标签页。

系统会提醒：

- 图片是否可能过度性感
- 标题是否有夸大词
- 材质是否填写完整
- 尺码表是否缺失
- 包装数量是否清楚
- 是否涉及敏感成人表达
- 是否需要更谨慎的模特图表达

### 内衣类目图片建议

- 优先使用平铺图、产品细节图、包装数量图
- 模特图使用自然站姿，避免挑逗姿势
- 胸贴、塑身衣、无痕内裤类目要特别注意主图尺度
- 不要使用成人暗示场景
- 不要在标题或图片里使用夸张、低俗、成人化表达

合规检查只是提醒，不替代平台规则和人工审核。

## 9. Excel 导出

点击“导出”标签页。

可以下载：

| 文件名 | 内容 |
|---|---|
| product_score_report.xlsx | 选品评分报告 |
| quotation_sheet.xlsx | 报价测算表 |
| listing_output.xlsx | 商品上架资料 |
| aliexpress_choice_full_output.xlsx | 包含录入数据、评分、报价、商品资料、合规检查的总表 |

推荐工作流：

1. 先导出总表做内部存档。
2. 把 `quotation_sheet.xlsx` 发给报价负责人。
3. 把 `listing_output.xlsx` 给上架或 ERP 负责人。
4. 把 `product_score_report.xlsx` 给选品或采购复盘。

## 10. 推荐日常使用流程

1. 收集供应商报价和基础资料。
2. 录入采购价、包装成本、重量、组合件数、MOQ。
3. 人工查看竞品，录入竞品最低售价和主流售价。
4. 输入准备报给平台的预估供货价。
5. 查看选品评分，筛掉 C/D 或负毛利产品。
6. 对 A/B 产品查看利润报价。
7. 对建议报价产品导出报价表。
8. 对进入上架池的产品导出商品资料。
9. 人工复核合规检查和英文资料。
10. 提交报价或上架。

## 11. 常见问题

### Q1：为什么我的产品评分低？

常见原因：

- 供货价没有利润
- 竞品主流售价太低
- 尺码太复杂
- 容易退货
- 没有填写差异化卖点
- 重量或包装不适合轻小件
- 被标记为有合规风险

### Q2：采购价是填单个还是整套？

填单个产品采购价。

例如 3 条装内裤：

- 采购价 RMB：填单条价格
- 组合件数：填 3

### Q3：竞品最低售价和竞品主流售价怎么填？

竞品最低售价是市场上低价竞品的参考价。

竞品主流售价是你认为比较正常、销量较多、不是极端低价的价格。

如果只填一个，建议优先填主流售价。

### Q4：为什么建议报价和安全报价不一样？

建议报价区间低更偏目标毛利，适合正常报价。

安全报价毛利更高，适合希望保留利润空间的产品。

激进报价更低，适合为了拿单或测试平台接受度，但利润安全垫更薄。

### Q5：生成的英文标题可以直接上架吗？

可以作为初稿，但上架前建议人工复核：

- 是否超过平台标题长度要求
- 是否含有敏感词
- 材质、数量、尺码是否准确
- 是否和图片、包装一致

### Q6：这个工具会自动抓取速卖通竞品数据吗？

不会。

竞品价格需要你手动录入或通过 CSV / Excel 导入。这样更安全，也避免违反平台规则。

## 12. 维护规则

如果后期要调整评分和报价逻辑，主要修改：

```text
config.py
```

可调整内容包括：

- 评分权重
- A/B/C/D 等级阈值
- 默认国内运费
- 默认质检人工
- 损耗率
- 资金占用率
- 目标毛利率
- 安全报价毛利率
- 产品类型对应的英文文案和材质
- 合规风险词

修改后重新提交并推送到 GitHub，Streamlit Cloud 会自动重新部署。
