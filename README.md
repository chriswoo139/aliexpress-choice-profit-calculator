# 速卖通 AliExpress Choice 全托管利润计算器

一个基于 Python + Streamlit 的中文利润测算工具，适合跨境电商卖家快速判断速卖通 AliExpress Choice / 全托管模式下某个 SKU 是否有利润，并反推最低建议供货价。

## 功能概览

- 单品利润计算
- 目标供货价反推
- 批量 SKU Excel 测算
- 结果 Excel 导出
- 参数设置与默认值管理

## 适用场景

- 女士无痕内裤
- 文胸洗衣袋
- 内衣配件
- 文胸延长扣
- 防滑肩带扣
- 其他轻小件跨境产品

## 公网部署

### 方案一：部署到 Streamlit Community Cloud

适合想要一个长期可分享网页地址的场景。

1. 把本项目上传到 GitHub 仓库
2. 打开 Streamlit Community Cloud
3. 选择 `New app`
4. 选择你的 GitHub 仓库
5. Main file path 填写 `app.py`
6. 点击 `Deploy`

部署成功后，会得到一个可公开访问的网址。

当前项目已经包含部署所需基础文件：

- `requirements.txt`
- `runtime.txt`
- `.streamlit/config.toml`

### 方案二：临时公网分享

如果只是临时发给别人试用，可以使用 Cloudflare Tunnel、ngrok 等工具把本地 `8511` 端口临时映射到公网。

这种方式的特点：

- 不需要改代码
- 启动快
- 链接通常不是永久固定的
- 你的电脑必须保持开机并持续运行程序

## 计算逻辑说明

### 1. 单套基础成本

```text
单套基础成本
= 单件采购成本 × 每套件数
+ 包装成本
+ 贴标成本
+ 国内物流摊销
+ 入仓物流摊销
+ 质检人工摊销
```

### 2. 最终总成本

```text
最终总成本
= 单套基础成本
+ 损耗成本
+ 退供/滞销预提成本
+ 资金成本
+ 汇率损耗成本
```

其中：

```text
损耗成本 = 单套基础成本 × 损耗率
退供/滞销预提成本 = 单套基础成本 × 退供/滞销预提率
资金成本 = 单套基础成本 × 资金成本率
汇率损耗成本 = 单套基础成本 × 汇率损耗率
```

### 3. 利润指标

```text
单套净利润 = 平台供货价 - 最终总成本
单件净利润 = 单套净利润 / 每套件数
毛利率 = 单套净利润 / 平台供货价
投入产出比 = 平台供货价 / 最终总成本
```

### 4. 建议供货价

```text
最低供货价 = 最终总成本 / (1 - 目标毛利率)
```

### 5. 是否值得做判断

- 毛利率 >= 30%：可做
- 毛利率 >= 20% 且 < 30%：谨慎
- 毛利率 < 20%：不建议做

阈值也可以在“参数设置”页面调整。

## 批量测算 Excel 字段

### 必填字段

- 产品名称
- SKU
- 采购价
- 件数
- 包装成本
- 物流成本
- 供货价
- 损耗率
- 目标毛利率

### 可选字段

- 类目
- 币种
- 贴标成本
- 入仓物流摊销
- 质检人工摊销
- 退供/滞销预提率
- 资金成本率
- 汇率损耗率

如果可选字段未填写，程序会自动使用“参数设置”页面中的默认值。

- 毛利率
- 建议供货价
