# 速卖通全托管女性内衣选品与报价工具

一个本地运行的 Python + Streamlit 工具，面向速卖通 AliExpress Choice / 全托管供货场景，适用于女性内衣、无痕内裤、运动内衣、文胸延长扣、肩带防滑扣、胸贴、塑身衣等类目。

## 功能

- 上传 CSV / Excel 或在网页中手动录入产品数据
- 自动计算选品评分，并输出 A/B/C/D 等级
- 自动计算全托管供货报价下的真实成本、毛利和建议报价
- 自动生成英文标题、五点卖点、详情描述、属性、尺码表、包装信息和图片提示词
- 针对女性内衣类目输出合规检查提醒
- 导出 `product_score_report.xlsx`、`quotation_sheet.xlsx`、`listing_output.xlsx`
- 支持导出包含全部工作表的总表

## 安装

```bash
pip install -r requirements.txt
```

## 启动

```bash
streamlit run app.py
```

打开本地页面后，可以直接使用示例数据，也可以下载导入模板后批量上传。

## 输入字段

核心字段包括：

- 产品名称
- SKU
- 产品类型：无痕内裤 / 运动内衣 / 文胸延长扣 / 肩带防滑扣 / 胸贴 / 塑身衣 / 其他
- 采购价 RMB
- 包装成本 RMB
- 国内运费 RMB
- 质检人工 RMB
- 损耗率 %
- 资金占用率 %
- 单品重量 g
- 组合件数
- 供应商起订量
- 颜色数量
- 尺码数量
- 是否轻小件
- 是否容易退货
- 是否有合规风险
- 差异化卖点
- 竞品最低售价
- 竞品主流售价
- 预估平台供货价

竞品价格只支持手动录入或 CSV / Excel 导入，本项目不包含平台爬虫。

## 评分逻辑

评分满分 100 分，规则在 `config.py` 中维护：

- 市场需求：20 分
- 供应价优势：20 分
- 尺码稳定性：15 分
- 差异化卖点：15 分
- 物流包装友好度：10 分
- 退货风险：10 分，风险越高扣分
- 合规风险：10 分，风险越高扣分

等级阈值：

- A：85 分及以上
- B：70-84.9 分
- C：55-69.9 分
- D：55 分以下

系统会同时输出是否建议打样、是否建议报价、是否建议放弃。

## 报价公式

公式说明也集中写在 `config.py` 的 `FORMULA_RULES` 中。

```text
基础成本 = 采购价 RMB × 组合件数 + 包装成本 RMB + 国内运费 RMB + 质检人工 RMB
损耗成本 = 基础成本 × 损耗率
资金占用成本 = 基础成本 × 资金占用率
单件真实成本 = 基础成本 + 损耗成本 + 资金占用成本
单件毛利 = 预估平台供货价 - 单件真实成本
毛利率 = 单件毛利 / 预估平台供货价
按毛利率反推报价 = 单件真实成本 / (1 - 目标毛利率)
```

默认报价参数在 `config.py` 的 `PRICING_RULES` 中调整：

- 最低可接受毛利率
- 激进报价毛利率
- 目标报价毛利率
- 安全报价毛利率
- 默认国内运费、质检人工、损耗率、资金占用率

## 合规说明

本工具只做运营提醒，不替代平台规则和人工审核。女性内衣类目建议重点检查：

- 图片是否过度性感
- 标题是否有夸大词或成人敏感表达
- 材质是否填写完整
- 尺码表是否缺失
- 包装数量是否清楚
- 胸贴、塑身衣、无痕内裤等类目是否需要更谨慎的模特图表达
